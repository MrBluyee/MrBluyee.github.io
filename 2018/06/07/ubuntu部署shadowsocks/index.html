安装：

apt-get install python3-pip

pip3 install shadowsocks

编辑配置文件：

vim config.json

加入：

{

"server":"your server addr",

"server_port":server_port,

"local_address":"127.0.0.1",

"local_port":local_port,

"password":"your password",

"timeout":300,

"method":"rc4-md5",

"fast_open": false

}

保存。

启用：

ssserver -c /yourroad/shadowsocks/config.json -d start

关闭：

ssserver -c /yourroad/shadowsocks/config.json -d stop

openssl升级到1.1.0以上版本后，导致shadowsocks启动报undefined symbol: EVP_CIPHER_CTX_cleanup错误。

原因：

在openssl1.1.0版本中，废弃了EVP_CIPHER_CTX_cleanup函数。

解决方案：

用vim打开文件：

vim /usr/local/lib/python/dist-packages/shadowsocks/crypto/openssl.py

(该路径请根据自己的系统情况自行修改，如果不知道该文件在哪里的话，可以使用find命令查找文件位置)，

跳转到52行（shadowsocks2.8.2版本）。

进入编辑模式将第52行：

libcrypto.EVP_CIPHER_CTX_cleanup.argtypes = (c_void_p,)

改为：

libcrypto.EVP_CIPHER_CTX_reset.argtypes = (c_void_p,)

再次搜索cleanup（全文件共2处，此处位于111行），将：

libcrypto.EVP_CIPHER_CTX_cleanup(self._ctx)

改为：

libcrypto.EVP_CIPHER_CTX_reset(self._ctx)

保存并退出启动shadowsocks服务，问题解决。
