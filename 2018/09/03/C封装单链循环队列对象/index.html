queue->traverse(queue,printQnode,0);
int circular = 0 表示队列只从头到尾遍历Qnode一遍；
int circular = 1 表示队列循环遍历Qnode，所以上面程序运行最后进入循环，不断遍历Qnode。
在实现的visit函数中，返回值如果不为0则退出遍历。
上述代码实现的visit函数为printQnode，返回值为0；
